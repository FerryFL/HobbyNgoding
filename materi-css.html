CSS (Cascade Style Sheet) adalah sebuah bahasa untuk mengatur tampilan web sehingga terlihat lebih menarik dan indah.

Dengan CSS, kita dapat mengatur layout (tata letak), warna, font, garis, background, animasi, dan lain-lain.


Font

  • font-family : mengatur jenis font yang digunakan
  • font-size : mengatur ukuran font
  • font-weight : mengatur ketebalan font


Text Styling

  • color : memberi warna pada tulisan
  • text-align : mengatur format paragraf
  • text-decoration : mengatur dekorasi pada teks


Background

  • background-color : mengatur warna pada background
  • background-image : mengatur gambar yang akan digunakan pada background
  • background-position : mengatur posisi gambar pada background


Selector

  • id
  • class
  • elemen HTML


Pseudo-Class

  • :link : style default pada sebuah link (a yang memiliki href)
  • :hover : style ketika kursor mouse terletak di atas sebuah elemen
  • :active : style ketika sebuah link di klik
  • :visited : style ketika link pernah dikunjungi sebelumnya


Margin
Margin digunakan untuk menciptakan ruang di sekitar elemen, di luar batas yang ditentukan.
CSS memiliki properti untuk menentukan margin untuk setiap sisi elemen, yaitu:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left


Padding
Padding digunakan untuk membuat ruang di sekitar konten elemen, di dalam batas yang ditentukan.
CSS memiliki properti untuk menentukan padding untuk setiap sisi elemen, yaitu:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
