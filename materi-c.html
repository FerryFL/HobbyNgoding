Bahasa C adalah sebuah bahasa pemrograman yang diorientasikan pada struktural dan prosedural. Bahasa pemrograman ini dikembangkan pada 1972 di Bell Laboratories oleh Dennis Ritchie.

Pada awalnya, bahasa C dikembangkan untuk utilitas yang bisa berjalan pada platform Unix. Kini, bahasa C dipakai pada banyak aplikasi.

Penting bagi Anda untuk menyadari bahwa bahasa pemrograman C telah menjadi sumber inspirasi bagi banyak bahasa pemrograman lainnya, seperti C++, C#, Java, PHP, Javascript, dan sebagainya. Bahasa-bahasa tersebut memiliki sintaks yang serupa dengan bahasa C. Bahasa C sering disebut sebagai "Bahasa pemrograman pilihan Tuhan", yang berarti bahasa C dianggap sebagai bahasa pemrograman yang menjadi dasar bagi semua bahasa lainnya.

Di bahasa C, variabel merupakan sebuah identifikasi atau nama yang digunakan untuk menunjukkan lokasi penyimpanan data. Elemen ini memiliki peran penting dalam menyimpan nilai-nilai. Selain itu, Anda dapat mengubah dan menggunakan nilai variabel tersebut berulang kali. Ketika Anda menamai variabel, terdapat aturan yang harus diikuti. Nama variabel harus diawali dengan huruf atau garis bawah (underscore). Namun, setelah itu dapat terdiri dari huruf, garis bawah, dan juga angka. Selain itu, penting untuk memperhatikan perbedaan antara huruf kecil dan huruf besar dalam penulisan variabel. Bahasa pemrograman C sangat sensitif terhadap perbedaan tersebut.

1. #include
#Include merupakan bagian integral dalam struktur bahasa C yang digunakan untuk memasukkan atau mengimpor fungsi-fungsi yang telah didefinisikan dalam file header. File header adalah sebuah file yang berisi definisi fungsi yang telah dibuat sebelumnya. Tujuan dari file header ini adalah agar fungsi-fungsi tersebut dapat digunakan dalam program C lainnya. File header umumnya memiliki ekstensi .h, seperti contoh stdio.h. Dalam konteks ini, Anda telah menggunakan perintah #include untuk mengimpor file stdio.h. File ini berisi deklarasi fungsi-fungsi dasar yang diperlukan dalam pembuatan program, seperti printf(). File stdio.h umumnya telah disertakan dalam instalasi kompiler C seperti gcc.

2. int main()
Fungsi main() adalah fungsi utama dalam sebuah program yang akan dieksekusi pertama kali. Oleh karena itu, penting untuk selalu menyertakan fungsi main() dalam setiap program, seperti contoh berikut:

int main()
{
//...
return 0;
}

Kata "int" sebelum main mengindikasikan jenis tipe data yang akan dikembalikan oleh fungsi. Dalam fungsi main(), penting untuk menyertakan pernyataan return 0. Ini berarti bahwa setelah eksekusi fungsi main, akan dikembalikan nilai 0. Alternatif lain adalah menggunakan void, yang berarti fungsi tidak mengembalikan nilai (kosong). Jika menggunakan void, kita tidak perlu menuliskan pernyataan return di akhir fungsi. Hal ini karena void tidak mengembalikan nilai apapun.
