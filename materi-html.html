Tag dalam HTML:

Tag adalah code pada HTML, dan tag terbagi menjadi 2, tag yang berpasangan dan tag yang tidak berpasangan:

Contoh dasar tag yang ada pada HTML:

<html>
<head>
	<title></title>
</head>
<body>

</body>
</html>

Di atas adalah template awal tag pada HTML, dan kita akan membahasnya satu persatu.

Tag yang dapat berada di dalam head adalah:

  • Judul Halaman: <title></title>
  • CSS: <style></style>
  • Javascript: <script></script>
  • Metadata: <meta>

Tag yang dapat berada di dalam body adalah:

  • Teks (h1, h2, h3, h4, h5, h6): Digunakan untuk menyimpan heading dengan tiap level, dengan level 1 yang paling besar dan level 6 yang paling kecil
  • Pendukung teks (br, hr, ...): Format <br> digunakan untuk menambahkan garis baru, dan format <hr> digunakan untuk menambahkan garis horizontal
  • Gambar (img): Digunakan untuk menyimpan gambar
  • Hyperlink (a): Digunakan untuk menyimpan hyperlink
  • List (ul, ol, ...): <ul> digunakan untuk unorder list (biasanya berisi panah, titik, strip), <ol> digunakan untuk ordered list (biasanya berisi angka, abjad)
  • Tabel (table, thead, tbody)
  • Form (form, input, select, button)
  • Script (script)
  • Object (object)
  • Grouping (div, span)

Kita juga dapat menambahkan komentar tanpa membuat code kita mengalami error dengan menggunakan format <!-- komentar -->.

Struktur Tag:

Kita dapat menyusun sebuah tag dengan struktur demikian: namaTag atribut="nilai". Contohnya: <body bgcolor="blue">

Tag bisa saja memiliki banyak atribut. Contohnya: <body bgcolor="blue" id="myid" class="myclass">

Berikut merupakan atribut Global yang ada pada HTML:

  • accesskey: Untuk menentukan bagaimana elemen ini dapat diakses menggunakan keyboard
  • class
  • id: Digunakan untuk mengelompokkan sebuah tag menjadi id atau class tertentu
  • dir: Digunakan untuk menentukan arah tulisan atau direction
  • lang: Tag untuk bahasa (language)
  • style: Digunakan untuk menyisipkan inline CSS
  • tabindex: Digunakan untuk menentukan urutan untuk membuat form
  • title: Digunakan untuk memberikan judul pada elemen tertentu

Heading dalam HTML:

Berikut adalah format untuk heading dalam HTML:

<html>
<head>
	<title>HTML Heading</title>
</head>
<body>

	<h1>Judul Artikel</h1>
	<p>Heading terbesar sebagai dasar website</p>
	
	<h2>Sub Judul</h2>
	<p>Untuk menambahkan sub judul</p>

</body>
</html>

List dalam HTML:

<html>
<head>
	<title>HTML Heading</title>
</head>
<body>

	<h3>To do List</h3>
	<ol>
		<li>Bangun Pagi</li>
		<li>Sarapan</li>
		<li>Tidur Lagi</li>
	</ol>

</body>
</html>

Penggunaan Hyperlink:

<html>
<head>
	<title>HTML Heading</title>
</head>
<body>

	<p>Klik <a href="https://www.google.com/">disini</a> untuk mengetahui lebih lanjut</p>

</body>
</html>

Cara Menyimpan Gambar pada HTML:

<html>
<head>
	<title>HTML Heading</title>
</head>
<body>

	<img src="logo.png">

</body>
</html>
