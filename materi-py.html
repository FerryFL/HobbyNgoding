Python adalah bahasa pemrograman yang cukup populer dikalangan para programmer. Python dibuat oleh Guido van Rossum dan diterbitkan pada tahun 1991. Python bisa digunakan sebagai :
- Web Development
- Software Development
- Matematika
- Dsb.

Variabel & Tipe data
Variabel adalah wadah untuk menyimpan nilai data. Python tidak memiliki perintah untuk mendeklarasikan variabel. Variabel dibuat saat anda pertama kali memberikan nilai.
Python dapat menyimpan data dalam sebuah variable dengan berbagai tipe data seperti berikut :
- Tipe Text : str
- Tipe Numeric : int, float
- Tipe Bool : bool
- Tipe Sequence : list, tuple, range
Di atas merupakan tipe tipe data yang umum digunakan dalam python. Di luar dari itu, masih banyak lagi tipe data yang tersedia dalam python. Contoh :
X = “Age” //bertipe string atau str
Y = 17 //bertipe int

Operasi Aritmatika
Operator aritmatika digunakan dengan nilai numerik untuk melakukan operasi matematika umum Operator aritmatika dalam python terdiri dari + , - , * , / , % , ** , // Contoh :
print(2+20/2)

Operasi Perbandingan
Operator perbandingan digunakan untuk membandingkan dua nilai Operator perbandingan dalam python terdiri dari == , != , > , < ,>= , <= Contoh :
a=5
b=10
if(a!=b):
print(“Tidak Sesuai”)

Operasi Logika Operator
logis digunakan untuk menggabungkan pernyataan bersyarat Operator logis dalam python terdiri dari AND, OR, dan NOT Contoh :
a=5
b=10
if(a==10 AND b==10):
print(“Nilai sama”)

Operator Assignment
Operator Assignment digunakan untuk menetapkan nilai ke variabel. Operator Assignment dalam python terdiri dari=, +=, -=, *=, /=, %=, //=, **=, &=, |=, ^=,>>= , <<= Contoh :
x=9
x +=1
print(x)

Pengondisian
Sebuah pengondisian ditulis dengan menggunakan kata kunci if, elif, dan else. Untuk mendefinisikan kondisinya, kita dapat menggunakan operasi perbandingan dan dilengkapi dengan menuliskan isinya Contoh :
a=5 b=10 if(a < b):
print(“Nilai b lebih besar”)
elif(a> b):
print(“Nilai a lebih besar”)
else: print(“Nilai a dan b sama”)


Pengulangan
Dalam python, pengulangan dapat dilakukan dengan 2 cara, yakni menggunakan while dan for Dengan WHILE loop kita dapat mengeksekusi satu set pernyataan selama kondisinya benar. Contoh :
i = 1
while(i <=5):
print(i)
i+=1

Function
Function dalam python merupakan blok kode yang hanya akan berjalan saat ia dipanggil. Function dapat mengembalikan data sebagai hasilnya. Penggunaan Function ini cukup berguna, terlebih bagi para programmer yang membutuhkan blok kode yang sama namun digunakan berulang. Contoh :
def outputFunction():
print(“Hello World”)
